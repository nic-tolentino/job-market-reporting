## Executive Summary

### Solution
Add `visa_sponsorship` field to company data, implement filtering and UI badges.

- Differentiates platform from competitors

Add filter endpoint:

```kotlin
@GetMapping("/companies")
fun getCompanies(
    @RequestParam visaSponsorship: Boolean? = null
): ResponseEntity<List<CompanyDto>> {
    val companies = if (visaSponsorship != null) {
        companyRepository.getCompaniesWithVisaSponsorship(visaSponsorship)
    } else {
        companyRepository.getAllCompanies()
    }
    return ResponseEntity.ok(companies)
}
```

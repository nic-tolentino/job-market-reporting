**Last Updated:** March 9, 2026  
**Priority:** HIGH - Scalability  
**Estimated Effort:** 8-10 hours  
**Dependencies:** None (foundational infrastructure)  
**Status:** Ready for Implementation

## Executive Summary

### Problem
Currently, webhook processing and admin sync triggers run **synchronously**:
1. Apify webhook fires → `ApifyWebhookController` receives payload
2. Controller processes entire sync inline (30-120 seconds)
3. Response sent after completion

**Risks:**
- Cloud Run request timeout (15 min default)
- Memory limits during large syncs
- No retry mechanism for transient failures
- Poor user experience on admin dashboard (long wait times)
- Cannot scale horizontally (each request is monolithic)

### Solution
Introduce **Google Cloud Tasks** for asynchronous background processing:

```
Webhook → Validate → Queue Task → 202 Accepted (immediate)
                ↓
        Cloud Tasks Queue
                ↓
        Background Worker → Process Sync → Update BigQuery
                ↓
        Dead Letter Queue (if fails after 5 attempts)
```

### Impact
- Webhook response time: **30-120s → <500ms**
- Automatic retry logic with exponential backoff
- Prevents Cloud Run timeouts
- Enables horizontal scaling
- Better failure isolation and monitoring

## Architecture

### Component Overview

| Component | Purpose | Technology |
|-----------|---------|------------|
| `ApifyWebhookController` | Receives Apify webhooks, pushes to queue | Spring Boot REST |
| `CloudTasksService` | Creates and queues tasks | GCP Cloud Tasks API |
| `SyncTaskHandler` | Internal endpoint that processes tasks | Spring Boot REST |
| `CloudTasksConfig` | Security config for internal endpoints | Spring Security |
| `Cloud Scheduler` | Triggers scheduled syncs | GCP Cloud Scheduler |

### Queue Structure

```
tech-market-sync-queue (primary)
├── Max attempts: 5
├── Max retry delay: 1 hour
├── Max dispatches per second: 10
└── Dead letter queue: tech-market-sync-dlq

tech-market-sync-dlq (dead letter)
├── Max attempts: 1
```

```kotlin
package com.techmarket.service

import com.google.cloud.tasks.v2.*
import com.google.protobuf.ByteString
import com.fasterxml.jackson.module.kotlin.jacksonObjectMapper
import com.fasterxml.jackson.module.kotlin.writeValueAsString
import com.fasterxml.jackson.module.kotlin.readValue
import org.slf4j.LoggerFactory
import org.springframework.beans.factory.annotation.Value
import org.springframework.stereotype.Service
import java.nio.charset.StandardCharsets

@Service
class CloudTasksService(
    @Value("\${gcp.cloud-tasks.queue-name:tech-market-sync-queue}")
    private val queueName: String,

    @Value("\${gcp.cloud-tasks.location:australia-southeast1}")
    private val location: String,

    @Value("\${gcp.project-id}")
    private val projectId: String,

    @Value("\${app.base-url}")
    private val baseUrl: String,

    private val cloudTasksClient: CloudTasksClient,
    private val objectMapper: ObjectMapper
) {

    private val log = LoggerFactory.getLogger(CloudTasksService::class.java)

    /**
     * Queues a sync task for background processing.
     * Returns the task name for tracking.
     */
    fun queueSyncTask(payload: SyncTaskPayload): String {
        val queuePath = QueueName.of(projectId, location, queueName).toString()
        val handlerUrl = "$baseUrl/api/internal/process-sync"

        val taskJson = objectMapper.writeValueAsString(payload)
        log.info("Queueing sync task: correlationId=${payload.correlationId}, source=${payload.source}")

        val task = Task.newBuilder()
            .setHttpRequest(
                HttpRequest.newBuilder()
                    .setHttpMethod(HttpMethod.POST)
                    .setUrl(handlerUrl)
                    .setBody(
                        ByteString.copyFrom(
                            taskJson.toByteArray(StandardCharsets.UTF_8)
                        )
                    )
                    .putHeaders("Content-Type", "application/json")
                    // Add auth header for internal endpoint verification
                    .putHeaders("X-Cloud-Tasks", "true")
                    .putHeaders("X-Cloud-Tasks-Correlation-Id", payload.correlationId)
                    .build()
            )
            .build()

        val request = CreateTaskRequest.newBuilder()
            .setParent(queuePath)
            .setTask(task)
            .build()

        val createdTask = cloudTasksClient.createTask(request)
        log.info("Task queued successfully: ${createdTask.name}")
        
        return createdTask.name
    }

    /**
     * Payload for sync tasks.
     */
    data class SyncTaskPayload(
        val manifestUrl: String,
        val source: String,           // APIFY, ATS, MANUAL
        val country: String?,
        val triggeredBy: String,      // WEBHOOK, SCHEDULED, ADMIN
        val correlationId: String,    // For tracing/debugging
        val maxRetries: Int = 5       // Override default retry count if needed
    ) {
        companion object {
            fun fromWebhook(
                manifestUrl: String,
                country: String?,
                correlationId: String
            ): SyncTaskPayload {
                return SyncTaskPayload(
                    manifestUrl = manifestUrl,
                    source = "APIFY",
                    country = country,
                    triggeredBy = "WEBHOOK",
                    correlationId = correlationId
                )
            }

            fun fromAdmin(
                manifestUrl: String,
                country: String?,
                correlationId: String
            ): SyncTaskPayload {
                return SyncTaskPayload(
                    manifestUrl = manifestUrl,
                    source = "MANUAL",
                    country = country,
                    triggeredBy = "ADMIN",
                    correlationId = correlationId
                )
            }

            fun fromSchedule(
                manifestUrl: String,
                country: String?,
                correlationId: String
            ): SyncTaskPayload {
                return SyncTaskPayload(
                    manifestUrl = manifestUrl,
                    source = "SCHEDULED",
                    country = country,
                    triggeredBy = "SCHEDULED",
                    correlationId = correlationId
                )
            }
        }
    }

    /**
     * Gets task details by task name.
     */
    fun getTask(taskName: String): Task {
        val request = GetTaskRequest.newBuilder()
            .setName(taskName)
            .build()
        return cloudTasksClient.getTask(request)
    }

    /**
     * Deletes a task (for cleanup).
     */
    fun deleteTask(taskName: String) {
        val request = DeleteTaskRequest.newBuilder()
            .setName(taskName)
            .build()
        cloudTasksClient.deleteTask(request)
    }
}
```

```kotlin
package com.techmarket.webhook

import com.techmarket.service.CloudTasksService
import com.techmarket.service.CorrelationIdGenerator
import org.slf4j.LoggerFactory
import org.springframework.http.ResponseEntity
import org.springframework.web.bind.annotation.*
import java.util.UUID

@RestController
@RequestMapping("/api/webhooks")
class ApifyWebhookController(
    private val cloudTasksService: CloudTasksService,
    private val correlationIdGenerator: CorrelationIdGenerator,
    private val webhookSignatureValidator: WebhookSignatureValidator
) {

    private val log = LoggerFactory.getLogger(ApifyWebhookController::class.java)

    /**
     * Receives Apify webhooks and queues them for background processing.
     * Returns 202 Accepted immediately.
     */
    @PostMapping("/apify")
    fun handleApifyWebhook(
        @RequestBody payload: ApifyWebhookPayload,
        @RequestHeader("X-Apify-Signature") signature: String?
    ): ResponseEntity<Unit> {
        // 1. Validate webhook signature (existing logic)
        if (signature == null || !webhookSignatureValidator.validate(payload, signature)) {
            log.warn("Invalid webhook signature")
            return ResponseEntity.status(401).build()
        }

        // 2. Generate correlation ID for tracing
        val correlationId = correlationIdGenerator.generate()
        log.info("Received Apify webhook for manifest: ${payload.manifestUrl}, correlationId=$correlationId")

        try {
            // 3. Queue task for background processing
            val taskPayload = CloudTasksService.SyncTaskPayload.fromWebhook(
                manifestUrl = payload.manifestUrl,
                country = payload.country,
                correlationId = correlationId
            )

            val taskName = cloudTasksService.queueSyncTask(taskPayload)
            log.info("Queued sync task: taskName=$taskName, correlationId=$correlationId")

            // 4. Return 202 Accepted immediately
            return ResponseEntity.accepted()
                .header("X-Correlation-ID", correlationId)
                .header("X-Task-Name", taskName)
                .build()

        } catch (e: Exception) {
            log.error("Failed to queue sync task, correlationId=$correlationId: ${e.message}", e)
            // Still return 202 - task can be retried manually
            return ResponseEntity.accepted()
                .header("X-Correlation-ID", correlationId)
                .header("X-Error", "Task queued with warnings")
                .build()
        }
    }
}
```

```kotlin
package com.techmarket.api.internal

import com.techmarket.service.CloudTasksService
import com.techmarket.service.JobDataSyncService
import com.techmarket.service.CloudTasksMetrics
import org.slf4j.LoggerFactory
import org.springframework.http.ResponseEntity
import org.springframework.web.bind.annotation.*
import java.time.Duration

@RestController
@RequestMapping("/api/internal")
class SyncTaskHandler(
    private val jobDataSyncService: JobDataSyncService,
    private val cloudTasksService: CloudTasksService,
    private val metrics: CloudTasksMetrics
) {

    private val log = LoggerFactory.getLogger(SyncTaskHandler::class.java)

    /**
     * Internal endpoint for processing sync tasks from Cloud Tasks.
     * Protected by X-Cloud-Tasks header verification.
     */
    @PostMapping("/process-sync")
    fun processSync(
        @RequestBody taskPayload: CloudTasksService.SyncTaskPayload,
        @RequestHeader("X-Cloud-Tasks") cloudTasksHeader: String?,
        @RequestHeader("X-Cloud-Tasks-Task-Name") taskName: String,
        @RequestHeader("X-Cloud-Tasks-Correlation-Id") correlationId: String?
    ): ResponseEntity<Unit> {
        // 1. Verify request is from Cloud Tasks
        if (cloudTasksHeader != "true") {
            log.warn("Invalid Cloud Tasks request - missing header")
            throw AccessDeniedException("Invalid Cloud Tasks request")
        }

        val effectiveCorrelationId = correlationId ?: taskPayload.correlationId
        log.info("Processing sync task, correlationId=$effectiveCorrelationId, source=${taskPayload.source}")

        metrics.recordTaskStarted()

        try {
            // 2. Execute sync logic with timing
            val duration = metrics.recordSyncDuration {
                jobDataSyncService.executeSync(
                    manifestUrl = taskPayload.manifestUrl,
                    source = taskPayload.source,
                    country = taskPayload.country
                )
            }

            log.info("Sync completed successfully, correlationId=$effectiveCorrelationId, duration=${duration}s")
            metrics.recordTaskCompleted()
            
            return ResponseEntity.ok()
                .header("X-Correlation-ID", effectiveCorrelationId)
                .build()

        } catch (e: Exception) {
            log.error("Sync failed, correlationId=$effectiveCorrelationId: ${e.message}", e)
            metrics.recordTaskFailed()

            // 3. Check if this is the final retry attempt
            try {
                val task = cloudTasksService.getTask(taskName)
                val attemptCount = task.dispatchCount

                if (attemptCount >= taskPayload.maxRetries) {
                    log.error(
                        "Task moving to DLQ after $attemptCount attempts, " +
                        "correlationId=$effectiveCorrelationId, manifest=${taskPayload.manifestUrl}"
                    )
                    // Optionally: Send alert to Slack/email here
                } else {
                    log.warn(
                        "Sync failed (attempt $attemptCount/${taskPayload.maxRetries}), " +
                        "will retry. correlationId=$effectiveCorrelationId"
                    )
                }
            } catch (e: Exception) {
                log.warn("Could not check task status: ${e.message}")
            }

            // Re-throw to trigger Cloud Tasks retry
            throw e
        }
    }
}
```

```kotlin
package com.techmarket.config

import org.springframework.context.annotation.Bean
import org.springframework.context.annotation.Configuration
import org.springframework.security.config.annotation.web.builders.HttpSecurity
import org.springframework.security.config.annotation.web.configuration.EnableWebSecurity
import org.springframework.security.config.http.SessionCreationPolicy
import org.springframework.security.web.SecurityFilterChain
import org.springframework.security.web.access.expression.DefaultHttpSecurityExpressionHandler

@Configuration
@EnableWebSecurity
class SecurityConfig {

    @Bean
    fun securityFilterChain(http: HttpSecurity): SecurityFilterChain {
        http
            .csrf { csrf ->
                csrf
                    .ignoringRequestMatchers(
                        "/api/internal/**",
                        "/api/webhooks/**"
                    )
            }
            .authorizeHttpRequests { auth ->
                auth
                    // Internal endpoints - only accessible from Cloud Tasks
                    // Cloud Tasks requests include X-Cloud-Tasks header (verified in handler)
                    .requestMatchers("/api/internal/**").permitAll()
                    
                    // Webhook endpoints - public but validated in handler
                    .requestMatchers("/api/webhooks/**").permitAll()
                    
                    // Admin endpoints - require authentication
                    .requestMatchers("/api/admin/**").hasRole("ADMIN")
                    
                    // Public API endpoints
                    .requestMatchers("/api/**").permitAll()
                    
                    // Everything else requires authentication
                    .anyRequest().authenticated()
            }
            .sessionManagement { session ->
                session.sessionCreationPolicy(SessionCreationPolicy.STATELESS)
            }

        return http.build()
    }

    @Bean
    fun httpSecurityExpressionHandler(): DefaultHttpSecurityExpressionHandler {
        return DefaultHttpSecurityExpressionHandler()
    }
}
```

```kotlin
package com.techmarket.api

import com.techmarket.service.CloudTasksService
import com.techmarket.service.JobDataSyncService
import org.springframework.http.ResponseEntity
import org.springframework.web.bind.annotation.*
import java.util.UUID

@RestController
@RequestMapping("/api/admin")
class AdminController(
    private val cloudTasksService: CloudTasksService,
    private val jobDataSyncService: JobDataSyncService
) {

    /**
     * Manually trigger a sync (queued for background processing).
     * Returns immediately with task details.
     */
    @PostMapping("/sync/trigger")
    fun triggerSync(
        @RequestParam manifestUrl: String,
        @RequestParam(required = false) country: String?
    ): ResponseEntity<Map<String, String>> {
        val correlationId = UUID.randomUUID().toString()
        
        val taskPayload = CloudTasksService.SyncTaskPayload.fromAdmin(
            manifestUrl = manifestUrl,
            country = country,
            correlationId = correlationId
        )

        val taskName = cloudTasksService.queueSyncTask(taskPayload)

        return ResponseEntity.ok(mapOf(
            "status" to "queued",
            "taskName" to taskName,
            "correlationId" to correlationId,
            "message" to "Sync task queued for background processing"
        ))
    }

    /**
     * Get status of a queued task.
     */
    @GetMapping("/sync/status/{taskName}")
    fun getTaskStatus(
        @PathVariable taskName: String
    ): ResponseEntity<Map<String, Any>> {
        try {
            val task = cloudTasksService.getTask(taskName)
            
            return ResponseEntity.ok(mapOf(
                "taskName" to task.name,
                "state" to task.status.name,
                "dispatchCount" to task.dispatchCount,
                "scheduledTime" to task.scheduleTime.toString(),
                "createTime" to task.createTime.toString()
            ))
        } catch (e: Exception) {
            return ResponseEntity.notFound().build()
        }
    }

    /**
     * Cancel a queued task.
     */
    @PostMapping("/sync/cancel/{taskName}")
    fun cancelTask(
        @PathVariable taskName: String
    ): ResponseEntity<Map<String, String>> {
        try {
            cloudTasksService.deleteTask(taskName)
            
            return ResponseEntity.ok(mapOf(
                "status" to "cancelled",
                "message" to "Task cancelled successfully"
            ))
        } catch (e: Exception) {
            return ResponseEntity.internalServerError()
                .body(mapOf("error" to e.message ?: "Unknown error"))
        }
    }
}
```

```kotlin
    private val cloudTasksService: CloudTasksService,
    @Value("\${app.sync.manifest-url}")
    private val manifestUrl: String

    private val log = LoggerFactory.getLogger(ScheduledSyncScheduler::class.java)

    @Scheduled(cron = "0 0 2 * * *", zone = "UTC")
    fun runScheduledSync() {
        log.info("Scheduling daily sync via Cloud Tasks")

        try {
            val correlationId = UUID.randomUUID().toString()
            val taskPayload = CloudTasksService.SyncTaskPayload.fromSchedule(
                manifestUrl = manifestUrl,
                country = null,  // Process all countries
                correlationId = correlationId
            )

            cloudTasksService.queueSyncTask(taskPayload)
            log.info("Scheduled sync queued, correlationId=$correlationId")

        } catch (e: Exception) {
            log.error("Failed to queue scheduled sync: ${e.message}", e)
        }
    }
}
```

## Error Handling & Retry Logic

### Retry Configuration

| Attempt | Backoff | Cumulative Time |
|---------|---------|-----------------|
| 1 | 0s | 0s |
| 2 | 60s | 1 min |
| 3 | 120s | 3 min |
| 4 | 240s | 7 min |
| 5 | 480s | 15 min |
| DLQ | - | - |

### Transient Errors (Auto-Retry)

- BigQuery timeout
- Network connectivity issues
- Rate limiting from external APIs
- Temporary service unavailability

### Permanent Errors (Go to DLQ)

- Invalid manifest URL
- Authentication failure
- Data parsing errors
- Schema validation failures

### Error Handling Best Practices

```kotlin
try {
    jobDataSyncService.executeSync(...)
} catch (e: TransientException) {
    // Re-throw to trigger retry
    log.warn("Transient error, will retry: ${e.message}")
    throw e
} catch (e: PermanentException) {
    // Log and don't re-throw - task will not retry
    log.error("Permanent error, moving to DLQ: ${e.message}", e)
    throw e  // Still throw for DLQ processing
} catch (e: Exception) {
    // Unknown error - retry
    log.error("Unknown error: ${e.message}", e)
    throw e
}
```

## Appendix: Cloud Tasks API Reference

### Queue Names

```
projects/{project-id}/locations/{location}/queues/{queue-name}
```

Example:
```
projects/tech-market-reporting/locations/australia-southeast1/queues/tech-market-sync-queue
```

### Task States

- `READY`: Task is ready to be dispatched
- `RUNNING`: Task is being dispatched
- `DONE`: Task has been successfully executed
- `FAILED`: Task has failed after all retries

### HTTP Headers from Cloud Tasks

| Header | Description |
|--------|-------------|
| `X-Cloud-Tasks` | Custom header we add for verification |
| `X-Cloud-Tasks-Task-Name` | Full task name |
| `X-Cloud-Tasks-Correlation-Id` | Custom header for tracing |
| `Authorization` | OIDC token (if configured) |
| `X-Cloud-Tasks-Dispatch-Deadline` | Dispatch deadline |

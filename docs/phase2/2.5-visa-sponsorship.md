# Phase 2.5: Visa Sponsorship Tracking

**Status:** Proposed  
**Priority:** MEDIUM - High User Value  
**Effort:** 4-6 hours  
**Owner:** Backend + Frontend  

---

## Problem Statement

**Migrant workers cannot easily identify companies that sponsor work visas.**

This is a critical pain point for:
- International students graduating from NZ/AU universities
- Skilled workers looking to relocate from overseas
- Current visa holders seeking to switch employers

**Current State:**
- `data/companies.json` already has a `visa_sponsorship` field (boolean)
- Field is not displayed on company profiles
- No filtering by visa sponsorship status
- Data accuracy is unknown

---

## Solution Overview

1. **Audit and update** `visa_sponsorship` data in `companies.json`
2. **Display** visa sponsorship badge on company cards and profiles
3. **Add filter** to company search/listings
4. **Show visa info** on company profile page (types sponsored, notes)

 * Information about a company's visa sponsorship policies.

data class VisaSponsorshipInfo(
    val offered: Boolean,
    val types: List<String> = emptyList(),
    val notes: String? = null,
    val lastVerified: String? = null,  // ISO date string
    val source: String? = null
) {
    companion object {
        /**
         * Creates VisaSponsorshipInfo from legacy boolean format.
         */
        fun fromBoolean(offered: Boolean): VisaSponsorshipInfo {
            return VisaSponsorshipInfo(
                offered = offered,
                types = emptyList(),
                notes = null,
                lastVerified = null,
                source = null
            )
        }
    }
}

**File:** `backend/src/main/kotlin/com/techmarket/persistence/company/CompanyMapper.kt`

Update parsing to handle both formats:

```kotlin
fun parseVisaSponsorship(value: Value?): VisaSponsorshipInfo? {
    if (value == null || value.isNull) return null
    
    // Check if it's a boolean (legacy format)
    if (value.booleanValue != null) {
        return VisaSponsorshipInfo.fromBoolean(value.booleanValue)
    }
    
    // Parse as object (new format)
    return try {
        val json = value.jsonValue
        VisaSponsorshipInfo(
            offered = json.get("offered")?.booleanValue ?: false,
            types = json.get("types")?.repeatedValue?.map { it.stringValue } ?: emptyList(),
            notes = json.get("notes")?.stringValue,
            lastVerified = json.get("last_verified")?.stringValue,
            source = json.get("source")?.stringValue
        )
    } catch (e: Exception) {
        log.warn("Failed to parse visa_sponsorship: ${e.message}")
        null
    }
}
```

### Step 3: Add API Endpoint (30 min)

**File:** `backend/src/main/kotlin/com/techmarket/api/CompanyController.kt`

```kotlin
@RestController
@RequestMapping("/api/companies")
class CompanyController(
    private val companyRepository: CompanyRepository
) {
    
    /**
     * Get companies filtered by visa sponsorship status.
     */
    @GetMapping
    fun getCompanies(
        @RequestParam(required = false) visaSponsorship: Boolean?
    ): ResponseEntity<List<CompanyDto>> {
        val companies = companyRepository.getAllCompanies()
        
        val filtered = if (visaSponsorship != null) {
            companies.filter { 
                it.visaSponsorship?.offered == visaSponsorship 
            }
        } else {
            companies
        }
        
        return ResponseEntity.ok(filtered.map { it.toDto() })
    }
}
```

### Step 4: Frontend - Visa Sponsorship Badge Component (1-2 hours)

**File:** `frontend/src/components/VisaSponsorshipBadge.tsx` (new)

```tsx
import React from 'react';
import { ShieldCheck } from 'lucide-react';

interface VisaSponsorshipBadgeProps {
    offered: boolean;
    types?: string[];
    notes?: string;
    showTooltip?: boolean;
}

export const VisaSponsorshipBadge: React.FC<VisaSponsorshipBadgeProps> = ({
    offered,
    types = [],
    notes,
    showTooltip = true
}) => {
    if (!offered) {
        return null;  // Don't show badge if not offered
    }

    const tooltipContent = (
        <div className="text-sm">
            <p className="font-semibold mb-1">Visa Sponsorship Available</p>
            {types.length > 0 && (
                <ul className="list-disc list-inside mb-1">
                    {types.map((type) => (
                        <li key={type}>{type}</li>
                    ))}
                </ul>
            )}
            {notes && <p className="text-muted">{notes}</p>}
        </div>
    );

    return (
        <div 
            className="inline-flex items-center gap-1.5 rounded-full bg-green-50 px-2.5 py-1 text-xs font-medium text-green-700"
            title={showTooltip ? undefined : tooltipContent}
        >
            <ShieldCheck className="h-3.5 w-3.5" />
            <span>Visa Sponsorship</span>
            {showTooltip && (
                <Tooltip content={tooltipContent}>
                    <Info className="h-3 w-3 cursor-help" />
                </Tooltip>
            )}
        </div>
    );
};
```
